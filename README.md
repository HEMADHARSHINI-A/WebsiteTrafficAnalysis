# 📊 Website Traffic Analysis Dashboard

A modern and interactive Website Traffic Analysis Dashboard built using **HTML, CSS, JavaScript, Bootstrap, and Chart.js**. This dashboard provides insights into website performance, user engagement, traffic sources, bounce rates, and conversion metrics through visually appealing charts and KPI cards.

---

## 🚀 Project Overview

The Website Traffic Analysis Dashboard helps businesses and analysts monitor website performance by displaying key metrics and interactive visualizations in a responsive and user-friendly interface.

---

## ✨ Features

- 📈 Interactive Line Chart for Sessions vs Users
- 🥧 Traffic Source Analysis (Pie Chart)
- 🍩 Device Usage Distribution (Doughnut Chart)
- 📊 Bounce Rate by Channel (Bar Chart)
- 📉 Users by Traffic Channel
- 📋 Top Performing Pages Table
- 💎 Modern Glassmorphism UI
- 📱 Fully Responsive Design
- 🌙 Dark Purple Analytics Theme
